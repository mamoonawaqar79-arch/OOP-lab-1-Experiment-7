# Student Management System

## Description
A simple C++ program that stores and displays student information using Object-Oriented Programming (OOP) concepts.

## Features
- Enter student name
- Enter roll number
- Display student details
- Uses classes and objects

## Concepts Used
- Classes and Objects
- Data Members
- Member Functions
- Encapsulation

## Sample Input
```text
Name: Ali
Roll No: 101
```

## Sample Output
```text
Student Information
Name: Ali
Roll No: 101
```

## How to Run
```bash
g++ Student.cpp -o Student
./Student
```
